# LifePATH Website

Official React website for **LifePATH**, a narrative-driven Android simulation game designed as a supplementary learning tool for the **Ethics** subject.

This website serves as the official landing page where users can learn about the game, explore its features, and download the latest APK release.

---

## Tech Stack

- React.js (Create React App)
- HTML
- CSS
- JavaScript

No backend — this is a frontend-only site.

---

## Getting Started

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm start
```

Open your browser and visit:

```text
http://localhost:3000
```
